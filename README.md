# AuctionWar:- An Online Auction Platform

A full-stack online auction system built with React (Vite) frontend, Node.js/Express backend, and MySQL database.

## Project Structure

- `frontend/` - React frontend using Vite, handles UI and user interactions.
- `backend/` - Node.js Express API server, handles authentication, auctions, bids, and business logic.
- `database/` - MySQL database scripts and schema (local setup or migrations).

## Technologies Used

- **Frontend:** React with Vite for fast build and development
- **Backend:** Node.js with Express for RESTful API and server-side logic
- **Database:** MySQL for relational data storage of users, auctions, and bids
- **Authentication:** JSON Web Tokens (JWT) for secure user authentication and authorization
- **Real-time Communication:** Socket.io for live bid updates and notifications
- **File Upload Handling:** Multer middleware for image uploads
- **Password Security:** bcryptjs for password hashing and verification
- **Task Scheduling:** node-cron for auction completion scheduler
- **HTTP Client:** Axios for frontend-backend communication via REST API
- **Environment Configuration:** dotenv for managing environment variables and sensitive data


## Features

- User registration and login with JWT authentication
- Create, browse, search auctions
- Place bids with realtime updates using Socket.io
- User profile and bid management
